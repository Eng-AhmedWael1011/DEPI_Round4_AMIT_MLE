# 🚀 Machine Learning Track - DEPI Round 4

Welcome to my repository for the **Machine Learning Engineer (MLE) Track** as part of the DEPI Round 4 curriculum (AMIT). This repository serves as a centralized hub for all my code, assignments, mini-tasks, and end-to-end machine learning projects developed during the sessions.

## 📁 Repository Structure

The repository is modularly organized into several domains of Data Science and Machine Learning:

### 🐍 Python & Data Science Basics
- **`src/python/`**: Core Python programming, Object-Oriented Programming (OOP) concepts, and basics.
- **`src/DS/`**: Data Science basics including Linear Algebra, Numpy, Pandas, Preprocessing, and Data Visualization.

### 🤖 Machine Learning & Deep Learning
- **`src/ML/`**: Supervised and Unsupervised learning algorithms, session tasks, and foundational model development.
- **`src/DL/`**: Deep Learning workshops and neural network implementations.

### 👁️‍🗨️ Advanced Domains
- **`src/Computer_Vision/`**: Computer vision techniques and algorithms.
- **`src/NLP/`**: Natural Language Processing tasks and applications.

### ⚙️ MLOps & Tracking
- **`src/MLOPS/`**: Machine Learning Operations concepts and methodologies.
- **`src/MLflow_tutorials/`**: Tracking experiments, model registry, and lifecycle management using MLflow.

### 🏆 Capstone Projects
- **`src/projects/P2/` (Census Income Prediction App):** A production-ready full-stack web application predicting income levels utilizing CatBoost, Flask, React, and SHAP explainability. Feature-rich with an interactive dashboard and dark/light theme toggles.
- **`src/projects/CT project/` (CT Medical Feature Store):** An enterprise-grade pipeline for medical CT scan analysis utilizing a centralized feature store, MLflow registry, and FastAPI deployment.

## 🛠️ Tech Stack & Requirements
The project environments utilize robust machine learning and development tools. Check out `requirements.txt` for the full list. Key libraries include:
- **ML & Data:** `scikit-learn`, `catboost`, `pandas`, `numpy`, `imbalanced-learn`, `scipy`
- **MLOps:** `mlflow`
- **Web & UI Development:** `Flask`, `streamlit`, `customtkinter`, `React` (in specific projects)
- **Model Explainability:** `shap`

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```
2. **Install core dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Explore!** Navigate to the specific project or session folder to review code, notebooks, and applications. Some full-stack projects like P2 have their own specific instructions to run both backend and frontend layers.
